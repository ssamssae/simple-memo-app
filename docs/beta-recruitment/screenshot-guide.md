## 📸 찍을 3장

### 1️⃣ 메모 리스트 (Hero shot)
**왜:** 앱 첫인상. 채워진 상태가 "아 이런 앱이구나" 직관적.
**무엇:** 즐겨찾기 3~4개 + 일반 메모 5~6개, 스와이프 힌트 살짝 보이면 최고.
**샘플 콘텐츠 (복붙용):**

```
[즐겨찾기 섹션]
회사 WiFi 비밀번호: cafeteria2026
오늘 해야 할 것: 세탁물 맡기기, 우유 사기
책 인용 — "완벽하지 않아도 완성하라"

[일반 메모 섹션]
선릉역 김밥집 이름 확인 (새로 생긴 거)
엄마 생신 선물 아이디어 모음
운동 루틴: 스쿼트 3x10, 푸쉬업 3x12
코드 리뷰 피드백 정리
주말 영화 추천 — 그랜드 부다페스트 호텔
```

### 2️⃣ 메모 작성 중 (Action shot)
**왜:** "제목 없이 본문만" 차별점 시각화.
**무엇:** edit 화면, 텍스트 채워진 상태. 커서가 살짝 보이면 "지금 쓰는 중" 느낌.
**샘플 콘텐츠:**

```
2026년 Q2 목표

1. 메모요 정식 출시
2. 베타 피드백 반영해서 v1.1 준비
3. 개인 홈페이지 버전업

매주 월요일 아침에 진척 확인.
```

### 3️⃣ 스와이프 or 드래그 상호작용 (Delight shot)
**왜:** 정적 스크린샷으로는 "왜 이 앱을 써?" 설명이 약함. 제스처 순간을 포착하면 "오 재밌어 보이네".
**무엇:** 두 가지 중 택일 (또는 둘 다 찍고 하나 고르기)
- **A. 스와이프**: 리스트에서 메모를 옆으로 스와이프 → 즐겨찾기 고정 아이콘 노출되는 순간
- **B. 드래그**: long press 후 메모를 들어올려 다른 위치로 이동하는 순간

둘 중 A(스와이프)가 Reddit 썸네일에서 더 눈에 띔. 추천은 **A**.

## 📱 기기 / 해상도 / 설정

| 항목 | 값 |
|---|---|
| 기기 | **안드로이드 실기기** (에뮬레이터는 상태바/배터리 비현실적이라 비추) |
| 해상도 | 1080x2340 이상 (요즘 기기 기본) |
| 방향 | 세로 고정 |
| 테마 | **시스템 다크 모드** (앱 자체가 다크 전용이지만 시스템도 다크로 맞추면 상태바 톤 통일) |
| 상태바 | **Do Not Disturb 켜서 알림 아이콘 제거** → 깔끔 |
| 배터리 | 충전 중 아이콘 숨기기 → 유선 해제 후 찍기 |
| 시간 | 오전 10:00 ~ 오후 3:00 사이 표기 (너무 늦은 시각은 "야간작업중" 인상) |
| 캐리어 | 가능하면 WiFi 아이콘만 보이게 (모바일 데이터 숨김) |

## 🎯 촬영 절차

1. 앱 실행 전, 위 샘플 콘텐츠를 **메모 9~10개** 미리 입력 (즐겨찾기 3~4 + 일반 5~6)
2. Do Not Disturb ON, 배터리 아이콘/시간 정돈
3. **1번 샷**: 리스트 화면 스크롤 최상단 → 전원+볼륨다운
4. **2번 샷**: Q2 목표 메모 탭 → 본문 커서 위치시킨 채 캡처
5. **3번 샷**: 리스트로 돌아와 메모 하나를 살짝 스와이프한 **중간 상태**에서 캡처
   - 팁: 스와이프 애니메이션이 짧으니 미리 연습 2~3회. 녹화(스크린 레코더) 후 프레임 추출이 제일 확실.

## 🖼️ 후처리

### 옵션 A: Raw 스크린샷 그대로 (추천)
- Reddit/한국커뮤는 **꾸미지 않은 스크린샷이 더 신뢰** 감. 인디 개발자 톤과도 맞음.
- 그대로 imgur 업로드.

### 옵션 B: 디바이스 프레임 씌우기 (선택)
- 필요하다고 느낄 때만.
- 도구: [mockuphone.com](https://mockuphone.com) (무료, 업로드 → 프레임 입힘)
- 단점: 파일 사이즈 증가, "너무 마케팅" 톤. Play Store 제출용엔 Raw 권장.

## ☁️ 업로드 (imgur 익명)

1. [imgur.com](https://imgur.com) 접속 (로그인 불필요)
2. 우상단 "New post" → 3장 드래그앤드롭
3. 업로드 완료 후 각 이미지 우클릭 → "Copy image address"
4. 링크 3개 메모 (모집글에 바로 붙여넣기)

> **대안:** Reddit 게시 시 `Images & Video` 옵션으로 직접 업로드하면 imgur 안 거쳐도 됨.
> 한국커뮤(클리앙/뽐뿌/에펨)는 각 사이트 이미지 업로드 기능 사용.
