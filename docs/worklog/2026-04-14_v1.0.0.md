2026.04.14 작업일지 v1.0.0

어제는 밤부터 다음 날 새벽까지 이어진 하루였다. 세션 여러 개를 오가며 메모요 앱의 UI 리디자인과 배포, 그리고 텔레그램 봇을 클로드 코드에 붙이는 작업을 병행했다. 단일 프로젝트만 기록하던 기존 v1.0.0 이전 일지와 달리, 이번 버전부터는 그날 작업한 모든 세션을 하나로 합쳐 기록한다.

메모요 앱에서는 UI를 애플 느낌의 다크 투톤으로 전면 리디자인했다. 배경은 어두운 두 가지 톤으로 정리하고, 텍스트와 아이콘을 amber 포인트 컬러로 통일했다. 편집, 응원, 저장, 뒤로가기 버튼은 하트 배지 느낌으로 맞추고 좌우 균형을 다듬었다. 메모 목록은 둥근 카드 형태로 바꾸었고, 스플래시와 LaunchScreen 배경도 검은색으로 통일했다.

기능적으로는 복수 선택 삭제를 새로 넣었다. 좌상단 편집 버튼으로 선택 모드에 들어가 메모 여러 개를 고른 뒤 한 번에 지울 수 있게 만들었다. 삭제 전 확인 다이얼로그도 붙여서 실수를 막았다.

배포도 진행했다. 웹 빌드를 올리고, 버전을 1.0.0+5 로 올려 테스트플라이트에 새 빌드를 보냈다. 사용되지 않던 쿠퍼티노 아이콘 의존성은 정리했다.

텔레그램 봇 연동 쪽은 하루 중 가장 오래 붙잡은 일이었다. 새벽에는 채널스 기능이 아직 계정에 열리지 않은 상태여서 응답이 돌아오지 않았고, 서버 플래그가 걸려 있을 가능성을 염두에 두고 재시도를 이어갔다. 아침에는 새로 만든 MyClaude 봇을 기준으로 텔레그램 구성 스킬을 돌리고 토큰을 넣어 페어링 코드까지 받아냈다. 클로드 코드를 채널스 플래그로 띄우는 실행 방식도 확인했고, 낮에는 친구 초대 가능 여부와 보안 설정을 정리했다. 저녁과 밤에는 알림이 도착하지 않는 문제를 여러 차례 재현하고 Stop 훅 설정과 필터 방향을 손봤다.

이 과정에서 권한과 자동 진행 정책도 손봤다. 위험한 작업이 아니면 자동으로 진행하고 비가역 작업만 사전 확인하는 규칙을 자동 메모리에 기록해 두었다. 텔레그램 세션 Stop 훅에서 알림을 건너뛰는 동작도 의도된 방향이라는 점을 메모리에 박아 둬서, 앞으로 같은 고민을 반복하지 않게 정리했다.

남은 작업은 텔레그램 채널이 하루 단위로 알림을 안정적으로 밀어 주는지 조금 더 지켜보고, 메모요 앱 다음 버전 기획을 정리하는 정도다.
